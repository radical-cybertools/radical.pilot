Contributing
============

This section offers useful information to the developers that want to join the
RADICAL-Cybertools (RCT) community and contribute to RADICAL-Pilot (RP)
development. Here, developers can find both technical and organizational
information about how and when RP is released or the code style to adopt when
opening a pull request, but also how to behave when participating in the
community activities.
